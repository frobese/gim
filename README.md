Schema-based In-Memory Graph Database for Elixir.

Documentation is available on [hex.pm](https://hexdocs.pm/gim)

## Installation
The package can be installed by adding `gim` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:gim, "~> 1.0.0"}
  ]
end
```
